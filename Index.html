課程目的: 設計 HTML 表單，搭配 JS 程式，建構簡易之 FHIR JSON 醫護表單系統

說明:醫護表單繁多，又常更新修改，常造成系統開發及維護得負擔。 本課程使用簡單的 HTML 及 JS 技術，可快速發展標準化醫護表單，並利於及維護修改。

FHIR 表單開發步驟

  1. 設計 HTML 網頁表單
  2. 確立表單與 FHIR 資料對應
  3. 套用 JS FHIR JSON 產生及上傳程式
  4. 表單發布及加入醫療作業流程
  5. 整理表單維護手冊(修改更新)

入門課程內容(同學僅須學會 HTML 即可設計簡易醫護表單系統)

  • HTML 入門
  • HTML 表單入門
  • VS code 安裝 (HTML 及 JS 設計環境)
  • 醫護表單範例

進階參考課程

  • JS 程式入門
  • JSON 資料
  • 網頁表單產生 FHIR JSON 資料
  • FHIR 入門
  • 範例表單及程式
  • MI-TW 聯測

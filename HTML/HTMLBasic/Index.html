練習題目:人物或地點介紹網頁

 可介紹個人、寵物、閨密、同學、住家或學校附近特別景點。每位同學介紹之內容應不相同。

要求:

  • 網頁至少包含主題、文字概述、圖片、以及相關連結等內容。
  • 網頁圖片及檔案不要太大。
  • 圖片檔案來源須為相對放置或 URL，發佈後才看的到。
  • 請上 github 申請帳號，以利上傳發布練習作業。
  • 可在影像或文字說明當中包含超連結資訊。
  • 並使用表格包含圖片或連結，以此整齊排列內容。
